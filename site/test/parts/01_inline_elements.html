Combining inline elements

  • Emphasized text can contain bold, code and sample output, subscriptssub and superscriptssup, underlines, highlights, links, and strikethroughs.
  • Bold text can contain italics, code and sample output, subscriptssub and superscripts,sup underlines, highlights, links and strikethroughs.
  • Code spans can contain italics, bold, subscriptssub and superscripts,sup underlines, highlights, links, and strikethroughs.
  • Superscripts can contain italics,sup bold,sup code,sup underlines,sup highlights,sup linkssup and strikethroughs.sup
  • Subscripts can contain italics,sub bold,sub code,sub underlines,sub highlights,sub linkssub and strikethroughs.sub
  • Underlined text can contain italics, bold, code, highlights, links, and strikethroughs.
  • Highlighted text can contain italics, bold, code, subscriptssub and superscripts,sup underlines, links, and strikethroughs.
  • Links can contain italics, bold, code, highlights, and strikethroughs.
  • Struck-through text can contain bold, code, subscriptssub and superscripts,sup underlines, highlights, and links.
  • Keyboard input can contain italics, bold, subscriptssub and superscripts,sup underlines, highlights, links, and strikethroughs.
  • Small text can contain bold, italics, code and sample output, subscriptssub and superscripts,sup underlines, highlights, links, and strikethroughs.
